# 3D Карусель Карточек Откликов

## Описание
Интерактивная 3D карусель для отображения откликов специалистов на заказы. Компонент создает эффект стопки карточек с плавными анимациями переходов и современным дизайном в стиле мобильных приложений.

## Основные возможности
- **3D эффекты**: Карточки отображаются с перспективой и глубиной
- **Анимированные переходы**: Плавная смена карточек с эффектами поворота и движения
- **Интерактивная навигация**: Кнопки переключения и индикаторы прогресса
- **Адаптивный дизайн**: Работает на всех устройствах
- **Градиентные фоны**: Красивые цветовые переходы
- **Портфолио с альбомом**: Полноценная галерея работ специалиста
- **Real-time обновления**: Автоматическое получение новых откликов

## Технологии
- **React 18** с TypeScript
- **Framer Motion** для анимаций
- **Tailwind CSS** для стилизации
- **Supabase** для данных и real-time обновлений
- **Lucide React** для иконок

## Быстрый старт
1. Импортируйте компонент `JobApplicationsList`
2. Передайте необходимые пропы (jobId, jobStatus, selectedProId, onApplicationSelect)
3. Убедитесь, что подключены зависимости Framer Motion и Supabase

## Демо
Компонент используется на странице деталей заказа (`/job/{id}`) для отображения откликов специалистов в красивом 3D интерфейсе.
